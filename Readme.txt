如需使用本代码，只需要调用下面函数
1、播放采用的线程，Media类最好创建指针 
2、SetHWND();设置播放资源句柄，确保资源有效
3、OpenFile();设置播放文件路径
4、代码中有写log文件的部分，请屏蔽然后按照自己的方式添加log输出

5、添加了一个demo文件，IDE是 vs2010,主要代码在确定按钮上
